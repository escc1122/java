# 选择排序 Selection Sort
每輪找出最小的位置並互換

    [30, 2, 3, 5, 4]
    
    i=0 minIndex=1 [2, 30, 3, 5, 4]
    //最小值為2 位置在1

    i=1 minIndex=2 [2, 3, 30, 5, 4]
    //最小值為3 位置在2

    i=2 minIndex=4 [2, 3, 4, 5, 30]
    //最小值為4 位置在4

    i=3 minIndex=3 [2, 3, 4, 5, 30]
    //最小值為5 位置在3
